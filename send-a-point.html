Send a Point

How you can say Thanks / Right On / Good Job to someone.

Points increase a person's ability to request offers from others. The more points you have, the greater your reach. For a point to come into existence on Easyah, somebody has to send it to somebody else. And it all happens one point at a time.

There are two ways to send a point:

IN THE APP

  • Using the search bar at the top of the home screen, search for the username of the person to send a point to.
  • Select their name from the results to be taken to their profile page.
  • Click the Send Point button.

BY SMS TEXT MESSAGE

First, get the username of the person you are sending a point to. Send it, as a text, all by itself, to 720 735 5210.
username
Note that anyone can send points this way. You do not need to have an account in Easyah to send someone a point via SMS.

You can send points to anyone you like. However, there is a time delay if you are going to send multiple points to the same person. It averages out to about 3 points per month that you can send to any single person.
